# NumPy 성적 통계 프로그램

## 1. 프로그램 소개

사용자에게 학생들의 점수를 입력받고, NumPy를 활용하여
인원수, 평균 점수, 최고 점수, 최저 점수, 표준편차를 계산하는 프로그램이다.

숫자가 아닌 값을 입력하거나 0부터 100의 범위를 벗어난 점수를 입력하면
오류 내용을 출력하고 다시 입력받는다.

## 2. 사용한 외부 라이브러리

- 라이브러리 이름: NumPy
- 사용 버전: 2.5.1
- 사용 목적: 여러 점수를 배열로 처리하고 통계를 계산하기 위해 사용하였다.

## 3. NumPy를 선택한 이유

NumPy는 여러 숫자를 배열 형태로 관리할 수 있고,
평균, 최댓값, 최솟값, 표준편차 등의 수치 계산 기능을 제공한다.

파이썬 기본 반복문만 사용하는 것보다 통계 계산을 간단하고
명확하게 작성할 수 있어 NumPy를 선택하였다.

## 4. 라이브러리 설치 방법

터미널에서 다음 명령어를 실행한다.

```bash
python -m pip install numpy
```

## 5. 설치 버전 확인 방법

```bash
python -c "import numpy; print(numpy.__version__)"
```

## 6. 프로그램 실행 방법

`main.py`가 있는 폴더에서 다음 명령어를 실행한다.

```bash
python main.py
```

학생들의 점수를 하나씩 입력하고, 입력을 종료하려면 `q`를 입력한다.

## 7. 주요 기능

- 학생 점수 입력
- 인원수 계산
- 평균 점수 계산
- 최고 점수와 최저 점수 계산
- 표준편차 계산

## 8. 주요 NumPy 코드 설명

```python
score_array = np.array(scores)
```

파이썬 리스트를 NumPy 배열로 변환한다.

```python
score_array.mean()
```

입력된 점수의 평균을 계산한다.

```python
score_array.max()
score_array.min()
```

최고 점수와 최저 점수를 계산한다.

```python
score_array.std()
```

점수들이 평균을 기준으로 얼마나 퍼져 있는지 나타내는 표준편차를 계산한다.
